## How to Update Article Content

### Through Wagtail Admin

1. Navigate to: http://localhost:8000/admin/
2. Click **Pages** in the left sidebar
3. Click on **Resources**
4. Find and click on the article you want to edit
5. Update the **Body** field with new content
6. Click **Save draft** or **Publish**

### Rich Text Editor Features
- Bold, italic, underline text
- Headings (H2, H3, H4)
- Bullet and numbered lists
- Links and buttons
- Images
- Block quotes
- Code blocks

### HTML Editing (Advanced)
- For advanced users, you can use HTML in the body field
- Include inline styles for custom formatting
- Use the article's gradient colors for consistency
- Colors available: `{article.card_gradient_start}` and `{article.card_gradient_end}`
